# Tkiner İle Hesap Makinesi

- Python ve Tkinter kullanılarak geliştirilmiş basit bir masaüstü hesap makinesi uygulaması

## Özellikler
- Toplama, çıkarma, çarpma ve bölme işlemleri kullanılabilir.
- Son 5 işlem görüntülenebilir.

## Muhtemel Güncellemeler
- Logaritmik denklem hesaplanması
- Trigonometrik denklem hesaplanması

## Kullanılan Teknolojiler
- Python 3
- Tkinter
